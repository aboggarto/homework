# Команда	|#Описание
|---|---|
|*git init	|Создаёт новый локальный репозиторий.|
|*git clone <url>|	Клонирует существующий удалённый репозиторий и создаёт его локальную копию.|
|*git add <файл>|	Добавляет файл в индекс (staging area) для последующего коммита.|
|*git add . (или git add .)|	Добавляет все изменённые и новые файлы в текущем каталоге и его подпапках.|
|*git add -p|	Интерактивно выбирает части файла для включения в коммит.|
|*git commit -m| "Сообщение"	Фиксирует изменения с указанным сообщением.|
|*git commit -am| "Описание"	Одновременно добавляет изменения в уже отслеживаемых файлах и выполняет коммит.|
|*git status|	Показывает текущее состояние репозитория: неотслеживаемые файлы, изменения, готовые к коммиту, и т. д..|
|*git log|	Отображает историю коммитов в хронологическом порядке. С опциями можно фильтровать и форматировать вывод.|
|*git push|	Отправляет локальные изменения в удалённый репозиторий.|
|*git remote add <имя> <url>|	Добавляет удалённый репозиторий с указанным именем и URL.|
|*git remote|	Показывает список настроенных удалённых|
|*git rm <файл>|  удаляет файл из индекса и рабочей директории. |
|*git mv <старое_имя> <новое_имя>| переименовывает или перемещает файл. |
